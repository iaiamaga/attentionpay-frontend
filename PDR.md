# Project Brief: AttnPay Ecosystem

## 1. Project Overview
AttnPay is a high-impact financial and security ecosystem focused on protecting user attention and assets. The platform combines traditional financial management (PIX) with advanced Web3 capabilities (Crypto/Wallets) under a "Lumina Glass" design language—a sophisticated dark-mode aesthetic with glassmorphism, depth, and vibrant neon accents.

**Tagline:** "Sua atenção, seu poder." (Your attention, your power.)
**Brand Identity:** AttnPay uses a distinct eye-shaped metallic logo with circuit-like motifs, symbolizing security and the "Attention" economy.

## 2. Core Features
- **Multilingual Support:** Full interface availability in Portuguese (PT-BR) and English (EN).
- **Secure Authentication:** Biometric verification, 2FA, and secure login flows.
- **Financial Hub:** Real-time balance tracking, heritage evolution charts, and detailed transaction history (PIX, Crypto).
- **Web3 Integration:** "Conectar Wallet" feature supporting Phantom Wallet and WalletConnect.
- **Security Protocols:** End-to-end encryption, protected identity, and DDoS attack prevention.
- **Support Center:** AI-driven support assistant and technical help center.

## 3. User Journey & Sitemap
1.  **Loading (Pre-home):** Immersive entry point with branding and progress synchronization.
2.  **Welcome / Onboarding:** Intermediate screen to route users to either Login or Sign-up.
3.  **Authentication:**
    *   Login & 2FA.
    *   Sign-up (Cadastro) with field validation.
    *   Password Recovery (Esqueci a senha).
4.  **Core Dashboard:** Overview of system efficiency, wallet balance, and recent activities.
5.  **Financial Detailed:** Deep dive into transactions and patrimonial evolution.
6.  **User Profile:** Personal data, contact info, and connected social networks.
7.  **Settings (Configurações):** Account security, appearance (Dark/Light), language, and Wallet connection.
8.  **Support:** Help Center and Chat Support.

## 4. Visual Identity (Lumina Glass)
- **Primary Color:** #6c5ce7 (Deep Violet)
- **Accent Colors:** Cyan/Blue gradients, Gold (for Logo elements).
- **Surface:** Dark navy/black (#13121b) with glassmorphism (backdrop-blur).
- **Typography:** Inter (Sans-serif) for high legibility across financial data.
- **Components:** High-gloss cards, rounded interactive elements (ROUND_EIGHT), and subtle glow effects for active states.

## 5. Technical Requirements
- **Responsive Design:** Mobile-first architecture.
- **Accessibility:** High contrast ratios, screen-reader friendly labels, and motion-respecting animations.
- **Security:** "Secure Authentication V2.0" and "Neon-Core System Architecture."
